# Kickr Control

Program connects to a Wahoo Kickr or associated GATT (General ATTribute) device and allows for the control of that device.


### Commandline Interface

`kickr_control [scan|connect]`

### Connect Interface Sub-Menu

```
> resistance [up|down]

> sprint <time>

> display_stats [--loop]
```

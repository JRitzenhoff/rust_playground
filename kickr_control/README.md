# Kickr Control

Rough architecture for a program that would connect to a GATT (General ATTribute) device and control that device.


### Commandline Interface

`kickr_control [scan|connect]`

### Connect Interface Sub-Menu

```
> resistance [up|down]

> sprint <time>

> display_stats [--loop]
```
